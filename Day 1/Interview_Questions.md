# Interview Questions – Day 1

## 1. What is ERP?

**Answer:**

ERP (Enterprise Resource Planning) is a business management software that integrates different departments such as Sales, Purchase, Inventory, Accounting, HR, and Manufacturing into a single system.

The main purpose of ERP is to improve business efficiency by providing real-time data, reducing manual work, and helping management make better decisions.

---

## 2. What is Odoo?

**Answer:**

Odoo is an open-source ERP software that helps businesses manage their operations using integrated applications.

It provides modules such as:

- CRM
- Sales
- Purchase
- Inventory
- Accounting
- Human Resources (HR)
- Manufacturing (MRP)
- Website
- eCommerce
- Point of Sale (POS)

Odoo is highly customizable and suitable for small, medium, and large businesses.

## 4. What is the role of an Odoo Functional Consultant?

**Answer:**

An Odoo Functional Consultant acts as a bridge between the client and the technical team.

Their responsibilities include:

- Understanding business requirements
- Analyzing business processes
- Configuring Odoo modules
- Preparing BRD and FRD documents
- Conducting User Acceptance Testing (UAT)
- Training end users
- Supporting ERP implementation
- Coordinating with technical developers

---

## 5. What is the difference between ERP and CRM?

**Answer:**

| ERP | CRM |
|------|------|
| Manages the entire business | Manages customer relationships |
| Covers Sales, Purchase, Inventory, HR, Accounting, Manufacturing, etc. | Focuses mainly on Leads, Opportunities, Customers, and Sales |
| Used by all departments | Mainly used by Sales and Marketing teams |
| Improves overall business operations | Improves customer engagement and sales performance |

**Example:**

ERP manages the complete supermarket.

CRM manages only customers and sales opportunities.

---

## 6. What is the difference between a Functional Consultant and a Technical Developer?

**Answer:**

| Functional Consultant | Technical Developer |
|------------------------|--------------------|
| Understands business processes | Develops custom solutions |
| Configures Odoo | Writes Python code |
| Meets clients | Creates custom modules |
| Gathers requirements | Designs models and views |
| Trains users | Integrates APIs |
| Performs UAT | Fixes technical issues |

The Functional Consultant focuses on business needs, while the Technical Developer focuses on implementation through coding.

## 7. What are the phases of ERP implementation?

**Answer:**

The ERP implementation lifecycle consists of:

1. Requirement Gathering
2. Business Analysis
3. Solution Design
4. Odoo Configuration
5. Data Migration
6. User Training
7. User Acceptance Testing (UAT)
8. Go Live
9. Post-Implementation Support

Each phase ensures a successful ERP deployment.

## 9. What is centralized data?

**Answer:**

Centralized data means storing all business information in one common database instead of using separate files or software.

For example:

- Sales team
- Purchase team
- Inventory team
- Accounting team

All departments access the same database, ensuring consistent and up-to-date information.

**Benefits:**

- No duplicate data
- Real-time updates
- Better accuracy
- Easy reporting
- Improved collaboration
