# Day 1 – Introduction to ERP & Odoo Functional Consultant

## What is ERP?

ERP stands for Enterprise Resource Planning.

ERP is software that integrates all business departments into a single system so that information flows efficiently across the organization.

Instead of each department using separate software, ERP provides one centralized database.

---

## Why ERP is Needed

Without ERP:

- Duplicate data
- Manual work
- Excel dependency
- Communication gaps
- Stock mismatch
- Invoice delays
- Poor reporting

With ERP:

- Single database
- Real-time information
- Better productivity
- Faster decision making
- Improved customer service
- Better inventory management
- Accurate reports

## Who is an Odoo Functional Consultant?

An Odoo Functional Consultant is a business expert who:

- Understands client requirements
- Studies business processes
- Configures Odoo
- Trains users
- Tests the system
- Supports implementation

A Functional Consultant generally does not write custom code. Their focus is on business processes and system configuration.

---

## Difference Between Functional Consultant and Technical Developer

Functional Consultant

- Business-focused
- Configures Odoo
- Meets clients
- Creates documentation
- Conducts user training

Technical Developer

- Python
- PostgreSQL
- XML
- Custom modules
- API integration
- Reports
